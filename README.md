# Projects Detail

1. ### Student Management System (C)
A simple command-line program to manage student records. This project demonstrates dynamic memory management, file I/O, and basic data structures in C.


#### Features
* Add, view, and delete student records
* Store student ID, name, and marks for multiple subjects
* Calculate average marks and assign grades automatically
* Sort students by average marks and name
* Save and load data from a file
* Dynamic memory allocation
* Interactive CLI with basic animations

Files: main.c, model.c, model.h, storage.c, storage.h, ui.c, ui.h<br>
Compilation: GCC
