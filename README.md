# Projects Detail

### 1. Student Management System (C)
A simple command-line program to manage student records. This project demonstrates dynamic memory management, file I/O, and basic data structures in C.

#### Features
* Add, view, and delete student records
* Store student ID, name, and marks for multiple subjects
* Calculate average marks and assign grades automatically
* Sort students by average marks and name
* Save and load data from a file
* Dynamic memory allocation
* Interactive CLI with basic animations

Files: main.c, model.c, model.h, storage.c, storage.h, ui.c, ui.h


### 2. Stringbuilder
This simple program implements a StringBuilder-like program in C that replicated the behavior of fgets().<br>

Instead of relying on standard library function, the program manually reads characterd from the input buffer one by one and constructs a string dynamically. The collected characters are then stored and displayed back to the user.<br>

The goal of this project is to understand how string input works internally and how string can be constructed character by character.<br>

### Concepts practiced
This project focuses on following core C concepts:
* Charcter input handling
* Buffer reading
* Dynamic string construction
* Manual string termination using '\0'
* Simple memory management
